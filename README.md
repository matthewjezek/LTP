# LTP
zapoctova prace k LTP

Při spuštění `sudo docker compose up -d` nad souborem spustíme kontejner dockeru na pozadí.
Ve stejné složce se vytvoří složka **mods** k přikládání módů.

Aktuální nastavení spustí:
- MC Forge 3.8
- na portu 25565
- s potvrzením EULA
- maximální pamětí 10 GB
- ve verzi Minecraftu 1.20.1
- s možností vždy restartovat (při spuštění OS)
